# vue老版本安装2.0

## 1.安装node npm 或cnpm,并测试版本

cnpm -v 

npm -v

vue -V

## 2.全局输入：

cnpm i -g @vue/cli

cnpm i -g @vue/cli-init

## 使用vue-cli(老版本)

下载：vue init webpack demo0

进入项目：cd demo

启动：npm run dev
